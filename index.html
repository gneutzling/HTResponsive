Como tudo começou...

Fixed

  • Elementos possuem tamanho fixo.
  • A resolução não afeta a maneira com que a página é exibida.

Fluid

  • O tamanho dos elementos é definido em porcentagem.
  • O conteúdo se adapta de acordo com a resolução da tela.

Adaptive

  • Elementos com tamanho fixo.
  • Atende dispositivos específicos.

Design Responsivo

O que é design responsivo?

  • Construido usando grids fluidos.
  • Usa media queries para controlar o design e conteúdo.
  • Adapta-se melhor a todo tipo de resolução.

Foco

O importante não é o dispositivo, mas sim o usuário.

Break Points

  • 1200px - Desktop widescreen
  • 960px - Tablets (landscape)
  • 768px - Tablets (portrait)
  • 480px - Smartphone (landscape)
  • 320px - Smartphone (portrait)

Regra de ouro

Alvo ÷ Contexto = Dimensão


300px ÷ 960px = 31.25%

Tipografia

Usar fontes na medida EM.


body { font-size: 10px; } h2 { font-size: 3em; /* 30px */ }

Imagens

Vídeos

.video { position:relative; padding-bottom: 56.25%; height: 0; overflow: hidden; } .video iframe { position: absolute; top: 0; left: 0; width: 100%; height: 100%; }

Outros pontos

  • Grids

  • Tabelas
  • Touchscreen

Conclusão

  • Só media queries não seriam o bastante, seria impraticável criar para cada resolução existente.
  • Devemos criar poucas media queries e fazer o layout fluir entre os break points.
  • Para isso devemos eliminar os pixels nas medidas e usar % e EM.
